# Proyecto: Estadísticas Lotería de Navidad España

Este proyecto es una herramienta educativa creada en Python para calcular y entender las probabilidades de ganar diferentes premios en el Sorteo Extraordinario de Navidad.

## Contenido
- `loteria_navidad.py`: Script interactivo que pide al usuario cuántos décimos tiene y muestra sus probabilidades.

## Cómo ejecutarlo
Requisitos: Tener Python instalado.

1. Abre una terminal.
2. Navega a la carpeta del proyecto.
3. Ejecuta el comando:
   ```bash
   python loteria_navidad.py
   ```

## Datos utilizados (Sorteo 2025)
- **Total números**: 100.000 (00000 - 99999)
- **El Gordo**: 4.000.000€ a la serie (400k al décimo).
- **Probabilidad base**: 1 entre 100.000 para el primer premio.

## Conceptos de Programación
Este script demuestra:
- Uso de **Listas y Diccionarios** para estructurar datos.
- **Funciones** para encapsular lógica matemática.
- **Entrada/Salida** de datos por consola.
- Cálculos matemáticos básicos de probabilidad.
